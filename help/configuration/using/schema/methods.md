# elemento de métodos {#methods--element}


## Modelo de conteúdo {#content-model-11}

methods:==method

## Atributos {#attributes-11}

nenhuma

## Pais {#parents-11}

`<srcschema>`

## Derivados {#children-11}

método

## Descrição {#description-11}

Este elemento permite que você defina um elemento `<method>`. É obrigatório para declarar um método.

## Descrição do atributo {#attribute-description-11}

Este elemento não tem atributos.

## Exemplos {#examples-8}

```
<methods async="true"
...// definition of one or more <method
</methods>
```
