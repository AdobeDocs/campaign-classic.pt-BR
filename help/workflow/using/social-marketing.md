# Marketing social{#social-marketing}



>[!AVAILABILITY]
>
>`:warning:` Esse recurso só está disponível no Campaign Classic v7. [Saiba mais](../../social/using/about-social-marketing.md)

Os fluxos de trabalho detalhados abaixo são instalados com o módulo **Marketing Social** por padrão.

<table> 
 <tbody> 
  <tr> 
   <td> <strong>Rótulo</strong><br /> </td> 
   <td> <strong>Nome interno</strong><br /> </td> 
   <td> <strong>Descrição</strong><br /> </td> 
  </tr> 
  <tr> 
   <td> <span class="uicontrol">Cálculo das estatísticas do Facebook</span> <br /> </td> 
   <td> <span class="uicontrol">statsFacebook</span> <br /> </td> 
   <td> Esse fluxo de trabalho calcula estatísticas vinculadas às interações com os fãs do Facebook.<br /> </td> 
  </tr> 
  <tr> 
   <td> <span class="uicontrol">Sincronizar fãs do Facebook</span> <br /> </td> 
   <td> <span class="uicontrol">syncFacebookFans</span> <br /> </td> 
   <td> Esse fluxo de trabalho importa os fãs do Facebook para o Adobe Campaign todos os dias às 7:00 AM.<br /> </td> 
  </tr> 
  <tr> 
   <td> <span class="uicontrol">Sincronização de páginas do Facebook</span> <br /> </td> 
   <td> <span class="uicontrol">syncFacebook</span> <br /> </td> 
   <td> Esse fluxo de trabalho sincroniza páginas do Facebook com o Adobe Campaign todos os dias às 7:00 AM.<br /> </td> 
  </tr> 
  <tr> 
   <td> <span class="uicontrol">Cálculo das estatísticas do Twitter</span> <br /> </td> 
   <td> <span class="uicontrol">statsTwitter</span> <br /> </td> 
   <td> Este fluxo de trabalho calcula estatísticas vinculadas a retweets e visitas ao X (anteriormente conhecido como Twitter).<br /> </td> 
  </tr> 
  <tr> 
   <td> <span class="uicontrol">Sincronizar páginas do Twitter</span> <br /> </td> 
   <td> <span class="uicontrol">syncTwitter</span> <br /> </td> 
   <td> Este fluxo de trabalho importa seguidores do X para o Adobe Campaign todos os dias às 7h.<br /> </td> 
  </tr> 
 </tbody> 
</table>
